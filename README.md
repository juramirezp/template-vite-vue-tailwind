# Vite Template

## Vue2 + Tailwind

## Additional Tech

This template includes some tech from CDN:

- Nunito Font
- Axios
- Bch-icons

## VSCode Extensions

- Headwind
- Tailwind CSS IntelliSense
- Tailwind Documentation
- Vetur
- Vue Language Features (Volar)

## Installation

```sh
degit juramirezp/template-vite-vue-tailwind my-proyect
cd my-proyect
npm i
```

## Development

```sh
npm run dev
```

## Build

```sh
npm run build
```
